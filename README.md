# 🏠 Melbourne Housing & Cost of Living Dashboard

An interactive data dashboard analysing rental prices, liveability scores 
and cost of living across 40 Melbourne suburbs.

## Features
- 🗺 Interactive Melbourne suburb map
- 📈 Rent trend analysis with ML forecasting (Linear Regression)
- 💰 Cost of living breakdown by suburb
- ⭐ Liveability radar (transport, safety, walkability, amenities)
- 🤖 AI-powered suburb insights
- 🔍 Filter by bedrooms, budget and suburb type

## Tech Stack
Python · Dash · Plotly · Pandas · Scikit-learn · OpenPyXL

## Data
Simulated realistic data for 40 Melbourne suburbs.
Ready to replace with real data from data.vic.gov.au
